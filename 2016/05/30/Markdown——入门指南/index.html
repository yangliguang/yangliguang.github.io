Markdown 语法的简要规则

  • 标题

# 一级标题

## 二级标题

### 三级标题

以此类推，总共六级标题，建议在井号后加一个空格，这是Markdown的标准语法。

  • 列表

文字前加-或*为无序列表

加1. 2. 3.为有序列表，

符号和文字之间加空格。

内部列表用+，如下：

  • test

  • test1

  • test2

  • test

  • test

  • 引用

文字前加>大于号即可，如：

这里是引用

  • 图片与链接

插入链接与插入图片的语法很像，区别在一个 ！号

图片为：![]()

链接为：[]()

  • 粗体与斜体

粗体：**这是粗体**

斜体：*这是斜体*

  • 表格

生成表格比较累.

name sno class
yang 2014 147
shuai 2015 147
chao 2016 147
  • 代码框

用两个`把中间的代码包裹，使用tab键即可缩进。

  • 分割线

使用三个星号，* 或者三个横线 —

  • 文字删除线

使用~~文字删除线~~

预览效果：

文字删除线
